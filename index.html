HTML

HTML (HyperText Markup Language) — це мова розмітки гіпертекстових документів, яка визначає структуру та вміст веб-сторінок. Вона використовує теги для вказівки на те, як саме елементи сторінки мають бути представлені. HTML не є мовою програмування, а скоріше мовою опису структури.

Детальніше

CSS

CSS, що означає "Cascading Style Sheets", - це мова, яка використовується для стилізації веб-сторінок, визначаючи їх зовнішній вигляд, та структуру. Вона дозволяє розробникам керувати елементами веб-сайту, такими як кольори, шрифти, розміри та позиціонування, роблячи його візуально привабливим та зручним для користувачів.

Детальніше

JS

JavaScript (JS) — динамічна, об'єктно-орієнтована[5] прототипна мова програмування. Реалізація стандарту ECMAScript. Найчастіше використовується для створення сценаріїв вебсторінок, що надає можливість на боці клієнта (пристрої кінцевого користувача) взаємодіяти з користувачем, керувати браузером, асинхронно обмінюватися даними з сервером, змінювати структуру та зовнішній вигляд вебсторінки.
